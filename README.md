# AI Code Reviewer

# Система статического анализа и аудита кода (LLM-based)

Инструмент для автоматизированного анализа Python-кода с использованием локальной языковой модели DeepSeek-Coder. Поддерживает работу с локальными файлами и удаленными репозиториями GitHub.

## Технические требования
1. Python 3.9 или выше.
2. Установленная среда Ollama.
3. Загруженная модель deepseek-coder:
   
   ollama pull deepseek-coder

# 1. Установка зависимостей
	pip install -r requirements.txt

# 2. Запуск серверной части
Сервер обрабатывает логику разбиения кода на части и взаимодействие с моделью через LangChain.
	
	
	python server.py
	
Сервер запустится на http://0.0.0.0:8000. Не закрывайте терминал во время работы.

# 3. Анализ
Локальный файл - 
	
	
	python cli.py "C:\path\to\script.py"
	
Ссылка на GitHub - 
	
	
	python cli.py https://github.com


Логика работы системы
Парсинг: Модуль ast сегментирует код на классы и функции.
Ограничение контекста: При превышении лимита в 150 строк блок кода автоматически дробится на части для корректной обработки моделью.
Клонирование: Для анализа GitHub используется временная директория temp_review, которая очищается перед каждым новым запросом.
Таймауты: В клиенте установлен таймаут 1200 секунд (20 минут) для обработки крупных проектов.
